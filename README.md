# Tic Tac Toe

Prosta aplikacja Tic Tac Toe napisana w czystym HTML/CSS/JS.

Uruchomienie:

1. Otwórz plik `index.html` w przeglądarce (dwuklik lub `Plik -> Otwórz`).

Funkcje:

- Kliknij komórki aby postawić `X` lub `O`.
- Aplikacja wykrywa zwycięstwo i remis.
- Kliknij `Restart`, aby zacząć od nowa.

Sugestie rozbudowy: gra przeciwko AI, licznik wyników, animacje.
